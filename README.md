# 🌍 Eco Travel Carbon Tracker

A React-based dashboard to visualize tourism carbon emissions and promote sustainable travel decisions.

---

## 🎯 Objective

This project aims to analyze and visualize carbon emissions generated by different modes of transportation such as flights, cars, and trains. It helps users understand the environmental impact of their travel choices and encourages sustainable tourism.

---

## 🚀 Features

* 📊 Interactive dashboard for carbon emission analysis
* ✈️ Comparison of transport modes (Flights, Cars, Trains)
* 📈 Emission trends over time
* 🌍 Region-wise insights (if data available)
* 🎛️ Filters for dynamic data exploration
* 🌱 Suggestions for eco-friendly travel

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5 & CSS3
* Chart libraries (Chart.js / Recharts)
